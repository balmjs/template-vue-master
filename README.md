# BalmJS template - vue-master
> A Vue project with router & vuex

## Installation

```sh
npm install -g balm-cli
balm init vue-master my-project

cd my-project
yarn install
# or
npm install
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](http://balmjs.com/)
