# BalmJS template - vue-master

> A Vue project with router & Vuex

## Installation

```sh
npm install -g balm-cli
balm init vue-master my-project

cd my-project
yarn
# or
npm i
```

> (Recommended) [BalmUI Lite](http://mdl.balmjs.com/) for your project

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](http://balmjs.com/)
